== Description ==

Well, this plugin was made specially to help people using wordpress insert adsense or any other
advertisement in the wordpress blogs in whatever position[1] in whatever post[2] they wished.

[1] above and below content, above and below post titles
[2] positions as in below post 2, above post 4, below post 5 whatever position does the blogger wish.

1. Easy "Copy & Paste" your ads code for embedding AdSense in your WordPress posts
2. Set the number and types of ads for page
3. Select the section of your blog where to insert each ad (homepage, posts, pages, categories, archives and tags)
4. All settings configured through WordPress Options interface (no knowledge of plugins or PHP required)
5. Easily test different ad formats and positions modifying your existing ads
6. Advertisements can be insterted in the sidebar [seperate feature]

== Installation ==

1. Upload the entire `cr-adsense-custom-placement` folder to the `/wp-content/plugins/` directory
2. DO NOT change the name of the `cr-adsense-custom-placement` folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create Advertisement with descried options

TIP: I have included a style.css, it is better that you wrap your advertisement in the <div class="ads"></div> codes
